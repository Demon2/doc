# Vue 是什么
Vue.js 是一套构建用户界面的渐进式框架。与其他重量级框架不同的是，Vue 采用自底向上增量开发的设计。Vue 的核心库只关注视图层，它不仅易于上手，还便于与第三方库或既有项目整合。
什么是渐进式框架,向上增量开发: 会在之后的文档中体现 【Vue 2.0，渐进式前端解决方案】
